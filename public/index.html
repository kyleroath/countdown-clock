about this project

what is this?

i created the ultimate countdown clock as a way to eliminate the need of constantly googling the release dates of my favorite games, movies, and shows.
it grew into something more, an opportunity to use vanilla javascript, jquery, bootstrap and libraries like momentjs & countdownjs.
and yes, everything is intentionally lower cased!

what is under the hood?

everything! originally the design idea was going to be nothing beyond javascript with momentjs, but quickly grew into much more. it now includes...

  • Javascript
  • jQuery
  • BootStrap 4
  • Momentjs
  • Countdownjs
if you'd like to take a look under the hood be sure to stop by my github repo for the project!

what else is being added?


as I continue to grow as a developer, ultimately I'd like for this project to be the ground works for a better optimized and implemented website. right now, it's completely static and uses localstorage to store information, if cache is cleared or the URL changes, this can be super problematic!

there will also be quality of life tweaks as it's used by friends and myself to track things. if you have great ideas for it, feel free to take the source and add on!
